## Installation

1. Clone the repository:

    ```sh
    git clone https://github.com/StepanBasharov/GoServerMonitor.git
    ```

2. Navigate to the project directory:

    ```sh
    cd GoServerMonitor
    ```

3. Install dependencies:

    ```sh
    docker-compose up --build
    ```
